# Base de Conhecimento — CostaWeb

Base de conhecimento estruturada para alimentar LLMs com contexto sobre o projeto CostaWeb.

## Estrutura

```
knowledge-base/
├── overview/          # Visão geral do projeto e stack tecnológico
├── architecture/      # Estrutura de pastas, roteamento e entry points
├── components/        # Documentação de cada componente React
├── styling/           # Design tokens, utilitários CSS
├── hooks/             # Hooks customizados
├── pages/             # Páginas legais e conteúdo
└── config/            # Configurações de build, lint e deploy
```

## Como Usar

Cada arquivo `.md` segue o padrão:

- **Descrição** — resumo curto do módulo
- **Localização** — caminho do arquivo-fonte
- **Responsabilidade** — o que o módulo faz
- **Detalhes Técnicos** — props, estado, dependências, padrões
- **Relacionamentos** — links para outros docs da base
